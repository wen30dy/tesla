Shade Surmise

The RGB color model is an additive color model in which red, green, and blue light are added together. Zero intensity for each component gives the darkest color (no light, considered the black), and full intensity of each gives a white.
RGB(RED-SHADE, GREEN-SHADE, BLUE-SHADE)

When one of the components has the strongest intensity, the color is a hue near this primary color (red-ish, green-ish, or blue-ish).

when two components have the same strongest intensity, then the color is a hue of a secondary color (a shade of cyan, magenta or yellow).

Hint

Adding red to green yields yellow;green to blue yields cyan; blue to red yields pink, all together yields white.
